# Sistema de Controle de Ordens de Serviço - INCRA/MA

Sistema desenvolvido para gerenciar e controlar as Ordens de Serviço do INCRA no Maranhão, com foco no acompanhamento de prazos e gestão de equipes.

## Funcionalidades

- 📊 **Dashboard**: Visualização de métricas, gráficos e próximos vencimentos
- 📋 **Listagem**: Visualização completa com filtros por situação, comunidade e tipo de ação
- ➕ **Cadastro**: Inclusão de novas ordens de serviço
- ✏️ **Alteração**: Edição de ordens existentes
- 🗑️ **Exclusão**: Remoção segura de registros
- 📥 **Exportação**: Exportação de dados para CSV

## Tecnologias Utilizadas

- Python 3.8+
- Streamlit - Framework para interface web
- SQLite - Banco de dados local
- Pandas - Manipulação de dados
- Plotly - Gráficos interativos
